객체 생성 및 제거

삽입메소드 part1

$(A).appendTo(B) : A를 B의 요소 내 뒷부분에 추가
$(A).prependTo(B) : A를 B의 요소 내 앞부분에 추가
$(A).insertAfter(B) : A를 B의 요소 뒤에 추가
$(A).insertBefore(B) : A를 B의 요소 앞에 추가

삽입메소드 part2

$(A).append(B) : B를 A요소 내 뒷부분에 추가
$(A).prepend(B) : B를 A요소내 앞부분에 추가
$(A).after(B) : B를 A 요소 뒤에 추가
$(A).before(B) : B를 A의 요소 앞에 추가

객체의 복사와 제거 메소드

clone() : 지정한 객체 (요소)를 복사할때 사용한다 remove()/ detach()/empty() : 객체요소를 제거하거나 내용을 비울떄 사용한다.
