CSS 기본선택자

* 선택자란 ?
특정한 HTML 태그를 선택할 때 사용하는 기능으로, 태그를 선택하여 원하는 스타일과 기능을 적용할 수 있다.

모든 선택자

html 문서 안의 모든 태그를 선택할 때 전체 선택자를 사용할 수 있다.

  • 테스트1
  • 테스트2
  • 테스트3

태그선택자

html 문서 내 같은 태그를 모두 선택할 때 사용한다.

태그명 {color : red;} 와 같은 형태로 사용한다.

아이디선택자

html 문서 내 해당하는 아이디 속성값을 가진 태그만 선택할 때 사용한다.

 #아이디명{color : red;} 와 같은 형태로 사용한다.

클래스 선택자

html 문서 내 여러 개의 태그를 동일한 클래스로 묶어 선택할 때 사용한다.

 #클래스명{color : red;} 와 같은 형태로 사용한다.
