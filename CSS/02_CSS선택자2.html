1. 기본속성선택자

기본선택자 뒤에 [] 를 사용하여 속성과 속성값을 사용

        선택자[속성=속성값]{
            스타일속성 : 값
        }

2. 자손선택자와 후손선택자

자손 : 바로 하위 요소 들 / 후손 : 하위 요소들 전부

        * 자손선택자
        a > b{
            스타일 속성 : 값;
        }
        * 후손선택자
        a b{
            스타일 속성 : 값;
        }

3. 동위 선택자

동위관계(같은 레벨) 에서 뒤에 위치한 태그를 선택하 때 사용

        * a 뒤에 있는 b 요소 하나만 선택

        a+b{
            스타일 속성 : 값;
        }

        * a 뒤에 있는 모든 b 요소 하나만 선택

        a~b{
            스타일 속성 : 값;
        }

4. 반응선택자

사용자의 움직임에 따라 달라지는 선택자

        *사용자가 클릭한 태그
        선택자:active{
            스타일속성 : 값
        }

        *사용자의 마우스가 올라가 있는 태그
        선택자:hover{
            스타일속성 : 값
        }

5. 상태선택자

입력양식의 상태에 따라 선택되는 선택자

        1) 체크 된 상태의 태그
        선택자:checked{
            스타일 속성 : 값;
        }
    
        2) 포커싱된 input태그
        선택자 : focus{
            설정내용
            
        }
    


        3) 사용가능한 input 태그 선택 : input 태그 : enabled{설정내용},
           사용불가능한 input 태그 선택 : input 태그 : disabled{설정내용}
