배열

배열이란 ?

자바스크립트에서는 자료형 지정이 없기 때문에 모든 자료형을 보관하는 변수의 모음을 배열로 취급한다.
자바에서의 ArrayList와 유사하다.

배열의 선언

배열 선언 시 크기를 지정하지 않고 선언할 수 있고, 크기를 정하여 선언할 수도 있다.

Array객체의 메소드

indesOf() : 배열에서 요소가 위치하는 인덱스를 리턴

concat(배열명) : 두 개의 배열을 결합할 때 사용한다

join() : 배열을 결합해서 문자열로 반환한다.

reverse() : 배열의 순서를 뒤집는다.

sort() : 배열의 오름차순 혹은 내림차순 정렬을 할 때 사용한다.

push() : 배열의 맨 뒤에 요소를 추가
pop() : 배열의 맨 뒤에 요소를 제거하여 리턴

shift() : 배열의 맨 앞 요소 제거
unshift() : 배열의 맨 앞 요소 추가

slice() : 배열의 요소를 선택해서 잘라내기
splice() : 배열의 index위치의 요소 제거 및 추가

toString() : 배열을 문자열로 반환한다.
