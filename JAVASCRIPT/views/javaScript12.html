객체

객체의 선언 및 호출

객체는 키값을 사용하여 속성을 식별한다.
중괄호{}를 사용해서 객체를 생성하고 속성에는 모든 자료형이 올수있다. 자바스크립트에서 객체는 프로퍼티의 집합 프로퍼티는 STRING 타입의 이름 과 속성리스트 를 가짐 프로퍼티는 "키" : "값" 의 형식으로 지정 

        var 객체 = {
            속성명 : 속성값,-- 프로퍼티
            속성명 : 속성값,
            ...

            key : value 형태!!
        };

객체의 식별자테스트

객체의 메소드 속성

객체의 속성 중에서 함수자료형인 속성을 메소드라고 한다.

객체와 반복문

객체의 속성을 살펴 볼때에는 단순 for문, while문은 접근 불가능하고 for in 문 사용

in 과 with 키워드

in : 객체 내부에 해당속성이 있는지 확인하는 키워드
with : 코드를 줄여주는 키워드, 호출 시 객체명을 생략할 수 있게 한다.





객체의 속성추가와 제거

처음 객체를 생성하고 이후 속성을 추가하거나 제거하는 것을 동적으로 할 수 있다.
