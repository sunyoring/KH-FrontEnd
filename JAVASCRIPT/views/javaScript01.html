자바스크립트개요

스크립트언어란?
사용자의 요구에 맞게 수행되도록
구현하는 언어, 매우 빠르고 쉽게 배울수 있으며 짧은 코드로 작성
할수있도록 고안되었다.
자바스크립트?
웹브라우저에서 가장많이 사용하는 인터프리터 방식의
객체지향언어, ECMA 스크립트 표준을 따르는 대표적인 웹 기술 구현 언어이다.
사용방법
  1. Inline 방식 :
    태그내에 직접 작성하는 방식 (간단한 소스 코드인경우 )
  2. Interal 방식 :
    Script 태그 를 별도로 선언하고 그내부에 코드를 작성하여 실행하는 방식
  3. external 방식 :
    외부의 .js 파일을 가져와 현재 웹문서에 등록하는 방식
주의할점
소스는 함수 단위로 작성하고, 이벤트 속성을 통해 실행한다.
일반적인 실행방식은 on 이벤트 명 = '실행할 함수명([전달인자])'

태그에 직접 스크립트를 작성하여 구동확인 (Inline 방식)

Script 태그를 활용하여 내부에서 작성한 소스 코드 실행하기 (Internal방식)

script 태그영역은 head 태그, body 태그 내에서 작성

별도로 작성한 자바 스크립트 함수 불러오기
