String 과 Math

String

자바스크립트의 내장 객체로 문자열 취급을 위해 사용하는 객체이다.
" " 또는 '' 로 묶여있는 값 (리터럴) String 이라는 자바스크립트 내장 객체가 존재하며 문자열과 관련된 기본적 메소드(함수) 제공

Math

정수, 부동소수점, Infinity, NaN 리터럴
Math라는 자바스크립트 내장 객체가 존재하며 수학과 관련된 기본적인 메소드를 제공함

infinity 와 NaN

대부분의 프로그래밍 언어는 0으로 숫자를 나누면 자동으로 오류가 발생하고 프로그램이 종료된다.
하지만 자바스크립트는 숫자를 0으로 나누면 infinity라는 값이 들어가게 된다.
또한 자바스크립트가 표현할 수 없는 숫자는 NaN(Not a Number)로 표시된다.
