폼관련태그

form 태그는 html 사용자가 입력할수 있는 양식을 제공하는 태그이다.
form태그내의 input태그들을 통해 사용자가 입력한 정보를 서버로 넘기면서 요청하는 역할을 한다.
* action 속성 : 폼내에 입력된 값들을 전달할 서버측 URL을 기입한다.
* method 속성 : get/post 방식으로 전송 방식을 지정한다.
> get : 서버에 사용자가 입력한 값을 전달하면서 요청시 사용자가 입력한 값들이 url에 노출되는 방식 (전달값에 제한이 있음) - 쿼리스트링
> post : 서버에 사용자가 입력한 값을 전달하면서 요청은 하되 url 에 노출되지않음 ( 전달값을 볼수 없고 , 크기제한 없음)

 

submit 버튼을 눌렀을 때 method 요청 전송방식으로 action에 지정한 경로로 사용자가 입력한 값을 (key = value) 형태로 전달한다.

fieldset : 폼 요소를 그룹으로 묶는 태그

legend : 묶는 폼 요소에 명칭을 붙이는 태그

필드셋으로 묶은 영역 별로 구분

text와 관련 된 input 태그들

type = "text"

-> 한 줄짜리 텍스트 입력을 할 수 있는 텍스트 상자이다.


type = "password"

-> 비밀번호를 입력할 수 있는 텍스트 상자이다.


type = "search", type="url", type="email", type="tel"

-> 겉모습은 텍스트 필드와 비슷하지만 각 각의 정보에 맞게 분화된 기능을 제공하는 텍스트 상자이다.





숫자와 관련 된 input태그

type = "number"

-> 입력창에 숫자를 입력하는 것이 기본이며, 브라우저에 따라 스핀박스가 표시되기도 한다.
(스핀박스 : 입력창 오른쪽 위 / 아래 화살표 버튼)


-> 슬라이드바를 통해 숫자 지정 가능


날짜/시간 관련된 input태그

type="date/month/week/time/datetime/datetime-local"
