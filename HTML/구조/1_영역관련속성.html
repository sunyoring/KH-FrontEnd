영역관련 속성

1) content영역 : 요소내의 내용 영역
2) border영역 : 요소의 테두리 영역
3) padding영역 : content와 border사이의 영역(여백)
4) margin영역 : 타 요소들간의 간격 영역 (바깥)
div의 테두리까지 포함한 가로길이
= width값 + 양쪽padding값 + 양쪽border값
= 300px + (10px * 2) + (20px * 2)
= 300px + 20px + 40px = 360px
div의 테두리까지 포함한 세로길이
= height값 + 상하padding값 + 상하border값
= 150px + (10px * 2) + (20px * 2)
= 150px + 20px + 40px = 210px
